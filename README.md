# ERP E-Commerce Pro

Professional E-Commerce Profit Calculator with Node.js backend + HTML frontend.

## Deploy Online (Free)

### Option 1: Render (Recommended)
1. Go to [render.com](https://render.com) → Sign up free
2. Click **New +** → **Web Service**
3. Connect your GitHub repo or upload files
4. Set **Build Command**: `npm install`
5. Set **Start Command**: `node server.js`
6. Click **Create Web Service**
7. Done! Your ERP is live with a real backend.

### Option 2: Railway
1. Go to [railway.app](https://railway.app)
2. New Project → Deploy from GitHub repo
3. Add variables: none needed
4. Deploy

### Option 3: Run Locally
```bash
npm install
npm start
```
Then open http://localhost:3000

## Features
- Real REST API backend
- JSON database (persistent)
- Multi-page professional architecture
- USD & DZD dual currency
- Auto-calculated KPIs: ROAS, CPR, RPC, Break Even
- Export CSV
- Charts & Analytics
