Formules Utilisées

• Commandes Livrées = Commandes × Confirmation% × Livraison%

• Revenus = Commandes × Prix de Vente

• Profit = Revenus − (Commandes × Coût Produit) − Dépenses Pub

• CPR = Dépenses Pub / Commandes

• CPR CAP = (Revenus / Commandes) − Coût Produit

• Break Even ROAS = (Revenus / Commandes) / CPR CAP

• Coût / Livrée = Dépenses Pub / Commandes Livrées

• ROAS = Revenus / Dépenses Pub

• RPC = Revenus / Nombre de Clicks
